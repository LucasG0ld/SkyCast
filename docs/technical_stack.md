# Technical Stack - SkyCast

## Frontend (Mobile)
- **Framework:** React Native (Expo SDK).
- **Language:** TypeScript (Strict mode).
- **Navigation:** React Navigation (Native Stack + Top Tab Navigator for swiping).
- **State Management:** Zustand (Persisted with middleware).
- **Styling:** StyleSheet (React Native) + `expo-blur`.
- **Animations:** `lottie-react-native` and `react-native-reanimated`.

## Frontend (Web - Landing Page)
- **Framework:** React (Vite.js).
- **Language:** TypeScript.
- **Styling:** Tailwind CSS or Styled Components.
- **Deployment:** Vercel or Netlify.

## Backend & Data
- **Primary API:** WeatherAPI.com (Free tier).
- **Geolocation:** `expo-location`.
- **Storage:** `@react-native-async-storage/async-storage`.
- **HTTP Client:** Axios.

## Development Tools
- **Package Manager:** npm or yarn.
- **Version Control:** Git (GitHub).
- **CI/CD:** Expo Application Services (EAS) for APK building.
